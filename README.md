# Quizitor

> Create and browse flashcards to study and learn faster

## Stack

- Next.js
- tRPC
- PostgreSQL with Prisma ORM

## Features

- [x] Authentication
- [x] Track quiz views
- [x] View public quizzes
- [x] Create quizzes
  - [x] Public/private
  - [ ] Delete
  - [ ] Make favorite
- [ ] Search quizzes
- [ ] ...
