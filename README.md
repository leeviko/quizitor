# Quizitor

> Create and browse flashcards to study and learn faster

## Stack

- Next.js
- tRPC
- PostgreSQL with Prisma ORM

## Features

- [x] Authentication
- [ ] Create quizzes
  - [ ] Delete
  - [ ] Make favorite
  - [ ] Public/private
- [ ] View public quizzes
- [ ] Search quizzes
- [ ] Track quiz views
- [ ] ...
